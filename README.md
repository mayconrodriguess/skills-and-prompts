# skills-and-prompts

Coleção de skills, prompts estruturados, fluxos reutilizáveis e guias para uso prático de IA no dia a dia técnico.

## Objetivo

Organizar prompts, skills e padrões de uso que apoiem automação, análise técnica, documentação e produtividade com IA.

## O que você vai encontrar aqui

- prompts reutilizáveis
- skills para fluxos técnicos
- estruturas de contexto
- guias de uso para agentes e LLMs

## Temas previstos

- troubleshooting assistido por IA
- documentação técnica
- organização operacional
- automação de tarefas com prompts

## Observação

O foco aqui é criar uma base prática e reaproveitável para uso de IA em cenários reais de T.I.
